playerSocketClosed

playerSocketClosed: ((player, payload) => void)

Type declaration

    • (player, payload): void

    • Emitted when a Node-Socket got closed for a specific Player. Usually emits when the audio websocket to discord is closed, This can happen for various reasons (normal and abnormal), e.g. when using an expired voice server update. 4xxx codes are usually bad.
